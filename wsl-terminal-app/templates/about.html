About WSL Terminal App

This application allows users to execute Linux commands through a web interface using WSL (Windows Subsystem for Linux).

Features

  • Execute Linux commands directly from your browser.
  • View command output in real-time.
  • Simple and user-friendly interface.

Installation

To run this application, ensure you have Flask and WSL installed on your Windows machine.

Usage

Navigate to the home page to start using the terminal interface.
